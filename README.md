# HostelHub 

HostelHub is a modern hostel discovery and management platform built with React.

The platform helps university students discover secure, affordable hostels near their campuses while giving landlords a powerful dashboard to manage properties, bookings, and tenants.

---

##  Current Features

- Responsive homepage
- Modern navigation
- Hero section
- Search bar
- Dynamic hostel cards
- Mobile-first design

---

##  Tech Stack

- React
- Vite
- Tailwind CSS
- React Icons

---

##  Planned Features

- Hostel search and filtering
- User authentication
- Student bookings
- Landlord dashboard
- Admin dashboard
- M-Pesa payments
- Reviews and ratings
- Favorites
- Hostel management SaaS
